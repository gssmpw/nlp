\section{Dataset Statistics}
\label{app:data}
We define gold location as the gold information span's end position divided by the total number of tokens in the input, which is uniformly distributed. We provide statistics for the Long datasets as shown below in \Cref{tab:dataset} and Short-form dataset in \Cref{tab:short_dataset_statistics}. Each dataset contains 15,360 training examples, 3,840 validation examples, and 4,800 test examples.
