\begin{abstract}
As scaling large language models faces prohibitive costs, multi-agent systems emerge as a promising alternative, though challenged by static knowledge assumptions and coordination inefficiencies. 
We introduce Knowledge-Aware Bayesian Bandits (KABB), a novel framework that enhances multi-agent system coordination through semantic understanding and dynamic adaptation. The framework features three key innovations: a three-dimensional knowledge distance model for deep semantic understanding, a dual-adaptation mechanism for continuous expert optimization, and a knowledge-aware Thompson Sampling strategy for efficient expert selection. Extensive evaluation demonstrates that our KABB achieves an optimal cost-performance balance, maintaining high performance while keeping computational demands relatively low in multi-agent coordination.
We will release the source code of our method in accordance with the review policy.

% Dynamic expert coordination is a fundamental challenge in multi-agent systems, especially in tasks requiring semantic understanding and adaptability. We presents the Knowledge-Aware Bayesian Bandits (KABB) framework. First, a three-dimensional knowledge distance model integrates concept overlap, dependency path optimization, and dynamic historical performance evaluation for precise expert-task matching. Second, a dual-adaptation mechanism leverages Bayesian updates with exponential time decay to ensure real-time adaptation to evolving tasks. Finally, a knowledge-aware Thompson Sampling strategy incorporates semantic distance into Beta distribution sampling, enabling efficient and accurate expert selection. Experiments demonstrate the framework’s ability to enhance response quality, offering a scalable solution for multi-agent systems in complex real-world scenarios.
