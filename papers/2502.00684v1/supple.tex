\section*{Atomic Concepts for LunarLander Environment}
We define atomic concepts based on different ranges of the 8-dimensional state space in LunarLander-v3. The state variables are: x-coordinate (X), y-coordinate (Y), x-velocity (Vx), y-velocity (Vy), angle (A), angular velocity (AV), right leg contact (RLeg), and left leg contact (LLeg).

\subsection*{X-coordinate Concepts}
\begin{itemize}
    \item X1: $-0.25 < x < 0$ (slightly left of center)
    \item X2: $0 \leq x < 0.25$ (slightly right of center)
    \item X3: $-0.4 < x \leq -0.25$ (moderately left)
    \item X4: $0.25 \leq x < 0.4$ (moderately right)
    \item X5: $x \leq -0.4$ (far left)
    \item X6: $x \geq 0.4$ (far right)
\end{itemize}

\subsection*{Y-coordinate Concepts}
\begin{itemize}
    \item Y1: $y \leq 0.1$ (very low altitude)
    \item Y2: $0.1 < y \leq 0.2$ (low altitude)
    \item Y3: $0.2 < y \leq 0.3$ (medium-low altitude)
    \item Y4: $0.3 < y \leq 0.4$ (medium altitude)
    \item Y5: $0.4 < y \leq 0.5$ (medium-high altitude)
    \item Y6: $0.5 < y \leq 0.7$ (high altitude)
    \item Y7: $y > 0.7$ (very high altitude)
\end{itemize}

\subsection*{X-velocity Concepts}
\begin{itemize}
    \item Vx1: $-0.1 \leq v_x < 0$ (low negative velocity)
    \item Vx2: $0 \leq v_x \leq 0.1$ (low positive velocity)
    \item Vx3: $0.1 < v_x \leq 0.2$ (slightly positive velocity)
    \item Vx4: $0.2 < v_x \leq 0.4$ (moderately positive velocity)
    \item Vx5: $0.4 < v_x \leq 1.0$ (high positive velocity)
    \item Vx6: $-0.2 \leq v_x < -0.1$ (slightly negative velocity)
    \item Vx7: $-0.4 \leq v_x < -0.2$ (moderately negative velocity)
    \item Vx8: $-1.0 \leq v_x < -0.4$ (high negative velocity)
\end{itemize}

\subsection*{Y-velocity Concepts}
\begin{itemize}
    \item Vy1: $-0.1 \leq v_y \leq 0.0$ (low downward velocity)
    \item Vy2: $-0.2 \leq v_y < -0.1$ (slightly downward velocity)
    \item Vy3: $-0.4 \leq v_y < -0.2$ (moderately downward velocity)
    \item Vy4: $-1.0 \leq v_y < -0.4$ (high downward velocity)
    \item Vy5: $0.0 < v_y \leq 0.2$ (slightly upward velocity)
    \item Vy6: $0.2 < v_y \leq 0.4$ (moderately upward velocity)
    \item Vy7: $0.4 < v_y \leq 1.0$ (high upward velocity)
\end{itemize}

\subsection*{Angle Concepts}
\begin{itemize}
    \item A1: $\theta \leq -1.0$ (extreme left tilt)
    \item A2: $-1.0 < \theta \leq -0.15$ (moderate left tilt)
    \item A3: $-0.15 \leq \theta \leq 0$ (slight left tilt)
    \item A4: $0 \leq \theta \leq 0.15$ (slight right tilt)
    \item A5: $0.15 < \theta \leq 1.0$ (moderate right tilt)
    \item A6: $\theta > 1.0$ (extreme right tilt)
\end{itemize}

\subsection*{Angular Velocity Concepts}
\begin{itemize}
    \item AV1: $\omega \leq -0.25$ (high leftward rotation)
    \item AV2: $-0.25 < \omega \leq -0.1$ (moderate leftward rotation)
    \item AV3: $-0.1 \leq \omega \leq 0$ (slow leftward rotation)
    \item AV4: $0 \leq \omega \leq 0.1$ (slow rightward rotation)
    \item AV5: $0.1 < \omega \leq 0.25$ (moderate rightward rotation)
    \item AV6: $\omega > 0.25$ (high rightward rotation)
\end{itemize}

\subsection*{Contact Sensor Concepts}
\begin{itemize}
    \item RLeg: Right leg contact sensor = 1
    \item LLeg: Left leg contact sensor = 1
\end{itemize}

\section*{Atomic Concepts for Blackjack Environment}
We define atomic concepts based on different components of the 3-dimensional state space in Blackjack. The state variables are: player's sum (P), dealer's visible card (D), and usable ace indicator (A).

\subsection*{Player's Sum Concepts}
\begin{itemize}
    \item P1-P21: Player's current sum equals the specified value
    \begin{itemize}
        \item P4-P11: Low sums (likely to hit)
        \item P12-P16: Medium sums (decision boundary region)
        \item P17-P21: High sums (likely to stick)
    \end{itemize}
\end{itemize}

\subsection*{Dealer's Card Concepts}
\begin{itemize}
    \item D1-D11: Dealer's visible card equals the specified value
    \begin{itemize}
        \item D2-D6: Low dealer cards (favorable for player)
        \item D7-D10: High dealer cards (unfavorable for player)
    \end{itemize}
\end{itemize}

\subsection*{Ace Concepts}
\begin{itemize}
    \item HasAce: Player has a usable ace (counts as 11)
    \item NoAce: Player does not have a usable ace
\end{itemize}
