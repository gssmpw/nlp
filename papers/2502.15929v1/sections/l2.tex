\section{\texorpdfstring{$\ell_2$ Mechanism}{L2 Mechanism}}
\label{sec:l2}
This section provides an algorithm for computing $\sigma$ to achieve an $(\eps, \delta)$-DP $\ell_2$ mechanism (\Cref{subsec:l2_dp}) and then describes a simple method for sampling the $\ell_2$ mechanism in parallel (\Cref{subsec:l2_sampler}). Without loss of generality, we assume that our statistic $T$ has $\ell_2$ sensitivity $\Delta_2 = 1$. If $\Delta_2 \neq 1$, we can run the algorithm on $T / \Delta_2$ and rescale.
