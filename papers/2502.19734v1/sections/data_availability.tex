\section{Data Availability Statement}
\label{sec:data_availability_statement}
We obtained speckle data from an LCLS experiment studying bacteriophage PR772, collected at the AMO instrument (experiment ID amo06156; run numbers: 90, 91, 94, 96, and 102). The data can be found at \url{https://www.cxidb.org/id-156.html}.
