\section*{Abstract}
%Obtaining a state-space representation from an identified input-output (IO) model 
%  for the use in controller synthesis
% linear parameter-varying
%  obtained using system identification 
%The dynamic and nonlinear dependency of the coefficient functions in the minimal state-space realization of an identified linear parameter-varying input-output representation complicates stability analysis and controller synthesis. 
A minimal state-space (SS) realization of an identified linear parameter-varying (LPV) input-output (IO) model usually introduces dynamic and nonlinear dependency of the state-space coefficient functions, complicating stability analysis and controller synthesis. 
%Specifically, whereas linear parameter-varying systems are often identified using an input-output model, analysis and controller synthesis usually requires a state-space realization.
The aim of this paper is to introduce and analyze a direct SS realization of this IO model that avoids this nonlinear and dynamic dependency, at the cost of introducing a nonminimal state.
It is shown that this direct SS realization 1) is reachable under a coprimeness condition on the coefficient functions of the IO model and a well-posedness condition on the model order, and 2) is never observable but that the unobservable directions converge to zero in a finite amount of steps, i.e., that the realization is reconstructible.
The derived results are illustrated through numerical examples in both the LPV and LTI case.
