We use the following instructions for the \io{}, \iocot{}, and \react{} baselines in both the synchronous and asynchronous datasets. Each method is few-shot prompted with an example optimal trajectory on a task unique to the dataset being evaluated on. Notably, the asynchronous example involves a time delay. We ensured that prompt formatting in both datasets was consistent. \react{} example observations are omitted to make experiments cheaper and because we did not note a major difference in performance during preliminary testing.
