\section{Data-Availability Statement}
\label{sec:data}
The artifact supporting this paper is available on Zenodo \cite{vimalaartifact} 
as well as in our GitHub repository \cite{vimalagit}.
It provides our framework in the F$^\star$ programming language that allows implementing 
MRDTs/CRDTs and automatically proving the verification conditions (VCs) required by our technique.
